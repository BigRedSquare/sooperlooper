Documentation ::: Midi Bindings

The engine can be controlled with MIDI via the ALSA sequencer interface on Linux.

You can edit/load/save the midi bindings through the MIDI Bindings window in the GUI, or do learn-style control binding by choosing the learn option from the context menu's of a button, slider or control. Control-Middle-clicking on the button or slider is a shortcut to start learning.

You can also load the intial bindings from the saved bindings file with the -m option (to the engine) or by specifying in the GUI's Looper Connection window. Example binding files are to be found in the source code directory with the extension '.slb'.
